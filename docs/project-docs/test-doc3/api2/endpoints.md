# エンドポイント一覧

## ユーザー

| メソッド | パス | 説明 |
|---------|------|------|
| `GET` | `/users` | ユーザー一覧取得 |
| `GET` | `/users/:id` | ユーザー詳細取得 |
| `POST` | `/users` | ユーザー作成 |
| `PUT` | `/users/:id` | ユーザー更新 |
| `DELETE` | `/users/:id` | ユーザー削除 |

## アイテム

| メソッド | パス | 説明 |
|---------|------|------|
| `GET` | `/items` | アイテム一覧取得 |
| `POST` | `/items` | アイテム作成 |
